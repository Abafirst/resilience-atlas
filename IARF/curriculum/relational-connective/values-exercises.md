# Relational-Connective Dimension: Values Exercises

> *"The most basic and powerful way to connect to another person is to listen. Just listen."*
> — Rachel Naomi Remen

---

## Overview

Values in the Relational-Connective dimension orient us toward the kind of relational person we want to be — not the relationships we want to have (goals), but the qualities we want to bring to all our relationships (values). This distinction is crucial: relational goals (finding a partner, maintaining a friendship) can be blocked by circumstances, but relational values (being honest, showing up with care, choosing vulnerability) can be expressed in any moment regardless of circumstance.

The exercises in this document help clients excavate, clarify, and act from their relational values within The Resilience Atlas™ framework. They are designed to be clinically rich, practically actionable, and deeply humanizing — honoring the full complexity of human connection while building the behavioral skills that sustain it.

## Exercise 1: My Relationship Values List

**Purpose**: Structured identification and prioritization of relational values
**Duration**: 20–30 minutes
**Modality**: Individual or group (small group version with sharing)

### Instructions

"Below is a list of relational values — qualities of how you show up in relationships. For each one, rate how important it is to you (1 = not important, 10 = very important). Then rate how consistently you're currently living this value in your relationships (1 = rarely, 10 = consistently)."

|---|---|---|---|

| Reliability and follow-through | | | |
| Warmth and affection | | | |
| Listening and understanding | | | |
| Expressing care and appreciation | | | |
| Appropriate vulnerability | | | |
| Setting and respecting limits | | | |
| Forgiving others | | | |
| Asking for help | | | |
| Supporting others through difficulty | | | |
| Being a consistent presence | | | |
| Celebrating others | | | |
| Maintaining connection across distance | | | |
| Accepting others as they are | | | |

"Now look at your ratings. Which 3 values have the largest gap? These are your priority areas for relational growth. Which 3 values are you already living well? These are your relational strengths to build from."

### Facilitator Notes
- Normalize gaps — they are not failures but opportunities
- Identify relational strengths first to build from existing foundation
- Use this tool at baseline and every 2–3 months as a progress indicator
- [Clinical example: A client who rated "expressing care and appreciation" as 9 in importance and 2 in current living identified this as their primary target — leading to a behavioral activation plan to send one specific appreciation message per week]

## Exercise 2: The Connection Inventory

**Purpose**: To map the client's current relational landscape — quantity, quality, and functional roles of connections
**Duration**: 30–40 minutes
**Modality**: Individual; requires quiet time for honest reflection

### Instructions

"This exercise helps you take stock of where you currently are relationally — not where you wish you were, but where you actually are.

In the table below, list the people who are currently part of your life. For each person, complete the columns honestly."

"Now step back and look at your inventory as a whole:
- How many genuinely supportive, two-way relationships do you have?
- Are there relationships that consistently drain rather than nourish?
- Are there people you've lost contact with whom you'd like to reconnect?
- What's one relationship that deserves more investment this week?"

### Facilitator Notes
- Reassure clients that a small number of high-quality connections is more protective than many low-quality connections
- Gently explore the quality ratings — some clients rate all relationships as high quality out of obligation
- Be sensitive to grief over lost relationships (death, estrangement, geographic separation)
- Cultural context matters: in collectivistic cultures, the network is likely larger and the individual's role within it is differently defined

## Exercise 3: What Kind of Friend/Partner/Family Member Do I Want to Be?

**Purpose**: To articulate relational identity values — who I want to be in relationships, not just what I want from them
**Duration**: 20–30 minutes
**Modality**: Individual (written reflection); can be shared in dyads

### Instructions

"I'd like you to think about the different relational roles in your life — friend, partner, family member, community member — and for each one, complete this reflection:

**As a friend, I most want to be someone who...**

**As a partner/intimate companion, I most want to be someone who...**

**As a family member (son/daughter/parent/sibling/etc.), I most want to be someone who...**

**In my community, I most want to be someone who...**

Write freely, without editing. Let the values that matter most to you come forward."

### Debrief Questions
- "Which of these descriptions felt most alive when you were writing it?"
- "Which felt most distant from who you currently are?"
- "If someone who knew you well read what you wrote — how much of it would they recognize in you?"
- "What would change this week if you acted more fully from these descriptions?"

## Exercise 4: Ripple Effect

**Purpose**: To connect relational values to broader community and social impact; to build a sense of interdependence and contribution
**Duration**: 20–25 minutes
**Modality**: Individual or small group

### Instructions

"Imagine dropping a pebble into a still pool of water. The ripples move outward — from the center, to the near shore, to the far shore. Our connections work the same way.

Draw three concentric circles. In the innermost circle, write the names of your closest relationships — the people most immediately affected by who you are and how you show up.

In the middle circle, write your broader social connections — community, colleagues, acquaintances.

In the outermost circle, write the broader world — your city, future generations, people you'll never meet but who your actions touch.

Now: how do the values you live in your innermost circle ripple outward? If you're more honest, more present, more connected with your closest people — how does that affect the middle circle? The outer circle?

And the reverse: when you're relationally isolated or contracted — who is affected beyond you?"

### Debrief Questions
- "What does this exercise suggest about why relational wellbeing matters beyond your own experience?"
- "Where in your ripple effect do you feel most responsible?"
- "What one relational action this week could have the biggest positive ripple?"

## Exercise 5: The Loneliness/Connection Continuum

**Purpose**: To normalize loneliness, explore its function, and connect movement toward connection to personal values
**Duration**: 15–20 minutes
**Modality**: Individual

### Instructions

"Loneliness exists on a continuum — from profound isolation to deep belonging. Most of us move back and forth on this continuum throughout our lives.

Place yourself honestly on this continuum right now:

**0 = Profound Isolation** ←——————————————→ **10 = Deep Belonging**

Your current position: ___

Now: what are the main factors that have moved you toward isolation in your life? And what has moved you toward belonging?

Notice that loneliness isn't a character flaw — it's information. It tells you that connection is a value, that your need for belonging is real and legitimate.

If your value is connection — and loneliness is the signal that you've moved away from it — what direction do you want to move? And what's the smallest step in that direction?"

### Cultural Considerations
- Loneliness and social expectation vary by culture; in some contexts solitude is valued and loneliness less stigmatized
- For clients from highly individualistic contexts, loneliness may carry extra shame; normalize the universality of the need
- For clients from collectivistic contexts, loneliness may be experienced as dissonant with cultural identity; explore this

## Exercise 6: Acts of Connection Commitment

**Purpose**: To translate relational values into specific weekly behavioral commitments
**Duration**: 10–15 minutes
**Modality**: Individual; ideal as a closing exercise or homework assignment

### Instructions

"Let's make this concrete. Based on the relational values we've been exploring, I'd like you to identify three acts of connection for the coming week — specific behaviors that express your relational values.

These should be:
- Specific (not 'be more connected' but 'call my sister on Wednesday')

## Exercise 7: Gratitude for Connection

**Purpose**: To strengthen awareness of existing connections through gratitude practice; to counter the negativity bias in relational experience
**Duration**: 10–15 minutes
**Modality**: Individual; can be adapted as a journaling practice

### Instructions

"Research shows that gratitude practices strengthen relationships — both because gratitude increases our awareness of relational gifts, and because expressing gratitude deepens connection with others.

For this exercise: think of 3 people in your life who have contributed to your wellbeing, growth, or survival — in ways large or small.

For each person, write or say:
- What they gave you
- What value of theirs you witnessed in what they gave
- What difference it made
- What you'd want them to know

[Optional]: Is there one of these people to whom you could express this gratitude directly — in a message, a call, or a conversation?"

### Facilitator Notes
- This exercise can be emotionally activating, particularly for clients who have lost significant relationships
- Allow space for grief alongside gratitude — they often coexist
- The optional direct expression step moves from values reflection to committed action
- Adapt for clients whose primary support has been non-human (spiritual figures, nature, animals)

## Cultural Considerations

| Exercise | Cultural Consideration |
|---|---|
| Relationship Values List | Some values (e.g., "setting limits") may conflict with cultural norms of deference; explore respectfully |
| Connection Inventory | In collectivistic cultures, the network may include ancestors and community in ways that transcend individual contact |
| What Kind of Friend/Partner... | Gender roles in relationships vary significantly; avoid prescriptive expectations |
| Ripple Effect | Resonates strongly with cultures that emphasize communal responsibility and interdependence |
| Loneliness Continuum | Solitude vs. loneliness is culturally constructed; explore the client's cultural framing |
| Acts of Connection | Adapt modalities of connection to culturally appropriate forms (e.g., communal meals, ritual gatherings) |
| Gratitude Practice | Gratitude expression is valued across cultures but forms differ significantly; adapt accordingly |
