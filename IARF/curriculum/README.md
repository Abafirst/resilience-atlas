# IARF Curriculum Overview

> *Curriculum is how the framework becomes practice — structured pathways for building resilience, dimension by dimension.*

---

## Purpose

The IARF Curriculum provides structured learning materials, intervention modules, and practice resources organized by resilience dimension. Each dimension folder contains materials that can be used flexibly — adapted for individual clients, group settings, workshops, or self-directed learning.

---

## How to Use the Curriculum

1. **Review the assessment results** from the Resilience Atlas™ to identify priority dimensions.
2. **Select the relevant dimension folder(s)** below.
3. **Review the dimension's curriculum README** for an overview of available materials.
4. **Select modules or activities** appropriate to the client's needs, goals, and context.
5. **Document using templates** from the `../templates/` folder.

---

## Curriculum Structure by Dimension

Each dimension folder follows the same structure:

```
[dimension]/
├── README.md          ← Dimension overview and curriculum guide
├── modules/           ← Structured intervention modules (coming soon)
├── activities/        ← Individual exercises and practice activities (coming soon)
├── worksheets/        ← Client-facing worksheets (coming soon)
└── resources/         ← Reference materials and handouts (coming soon)
```

## Guiding Principles for Curriculum Use

- **Individualization first**: Curriculum materials are starting points, not scripts. Adapt to each client.
- **Data-driven**: Curriculum selection and pacing should be guided by assessment data and session-level progress data.
- **Values-guided**: Connect curriculum activities to each client's personal values.
- **Trauma-informed**: All curriculum delivery follows trauma-informed principles (safety, choice, collaboration, empowerment).
- **Strengths-anchored**: Begin with what's already working in the target dimension; build forward.

---

## Integration with the IARF Methodology

The curriculum is used primarily during **Phase 3: Intervention** but informs all phases:

- **Phase 1 (Assessment)**: Curriculum familiarity helps clinicians recognize strengths and gaps
- **Phase 2 (Analysis)**: Curriculum content informs functional hypothesis development
- **Phase 3 (Intervention)**: Primary use — curriculum modules provide the "what" of intervention
- **Phase 4 (Monitoring)**: Curriculum completion and skill generalization are monitored through data
