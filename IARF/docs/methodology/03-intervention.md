# Phase 3: Intervention Design & Implementation

> *The best intervention is the one that's right for this person, in this context, at this moment — and that changes as they do.*

---

## Overview

The Intervention Phase translates the case conceptualization from Phase 2 into a structured, individualized action plan. IATLAS interventions are dimensional (targeting specific resilience domains), evidence-based (grounded in ABA and ACT), collaborative (developed with the client), and measurable (tied to observable outcomes).

**Goals of Phase 3:**
- Design a dimensional intervention plan tailored to the individual
- Select evidence-based strategies for each targeted dimension
- Establish clear, measurable goals and outcomes
- Implement interventions using ABA and ACT principles
- Build generalization and maintenance from the start

---

## Intervention Planning Principles

### 1. Dimensional Targeting
Interventions are organized by dimension. While work often touches multiple dimensions simultaneously, each session or module has a **primary dimensional focus** to maintain clarity and measurability.

### 2. Function-Based Design
Every intervention addresses the function identified in the analysis phase. An intervention that doesn't address the *function* of a behavior is unlikely to produce lasting change.

### 3. Strengths Anchoring
Every intervention plan begins by identifying **existing strengths** within each target dimension. Interventions build from what's already working.

### 4. Values-Guided Action (ACT)
Interventions are connected to the client's identified values. This provides intrinsic motivation and direction, especially when change is difficult.

### 5. Skill-Building + Environmental Design
IATLAS targets both:
- **Skill building** — teaching new resilience behaviors
- **Environmental modification** — arranging conditions that support those behaviors

## Intervention Plan Structure

Each IATLAS Intervention Plan includes:

| Component | Description |
|-----------|-------------|
| **Client Profile Summary** | Brief synthesis of assessment and analysis findings |
| **Priority Dimensions** | 1–3 dimensions targeted in this plan |
| **Long-Term Goals (LTGs)** | Broad, meaningful outcomes (3–6 month horizon) |
| **Short-Term Objectives (STOs)** | Specific, measurable steps toward each LTG |
| **Strategies & Procedures** | Specific interventions for each objective |
| **Generalization Plan** | How skills will be practiced across settings |
| **Data Collection Plan** | What will be measured and how |
| **Review Schedule** | When the plan will be formally reviewed and updated |

## ABA Implementation Standards

IATLAS intervention implementation follows ABA's seven dimensions (Baer, Wolf & Risley, 1968):

1. **Applied** — Socially significant goals
2. **Behavioral** — Observable, measurable targets
3. **Analytic** — Data-driven decision-making
4. **Technological** — Procedures described precisely enough to replicate
5. **Conceptually Systematic** — Linked to behavioral principles
6. **Effective** — Producing meaningful change
7. **Generality** — Change that lasts across time, settings, and people
